### UI Defects (intended functions and appearance)

- Misaligned Items: off-centre or elements in unexpected places  
- Overlapping elements: text spilling, stacked buttons
- Broken Links, icon or button: a non functioning button, wrong icon 
- Non-responsive design: breaking ui for different resolutions, zooming 
- Inconsistent styling: Theme mismatch, design mismatch, poor contrast, difficult to read
- missing alt texts: Alt text for visually impaired users
- pop -up malfunctions 
- Scrolling issues
- form validation defects 
- Lost states: a checkbox that resets 

### UX Defects (User experience: defects, errors, design shortcomings)

- Confusing Navigation: complex navigation can confuse user , vague labels, unclear icons, lack of logical structure
- Unclear feedback: lack of immediate and clear feedback messages, confusion about the action happened or not 
- Findability issues: missing breadcrumbs, poor search functions,
- Excessive interaction: Too many steps, overloaded menus, too much empty space, multi step process for simple tasks
- Jarring Visuals: poor fonts, colours, images and inconsistent animations
- Inconsistency
- Dead ends in the user journey 

### Security Defects
- Obsolete Code : deprecated libraries or functions, version mismatch
- Suspicious Open-source: possible backdoor, 
- SQL injection attacks : improper input validation
- Setup mis-configurations: OS, browse, http headers 
- Cross-site Scripting: malicious scripts 
- Buffer overflow
- Server side request forgery 
- Weak access control: strict role based system
- Feeble encryption: outdated encryption methods 

### Performance Defects
- High Memory usage
- Slow loading 
- System crashing under high load
- ineffective algorithms : excessive resources 
- Network latency: Understanding how the system reacts to different network conditions 
- Database performance issue : slow insertion, slow retrieval, reduced system response
